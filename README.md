# Responsive Navigation Bar

A simple mobile-first responsive navigation bar built using HTML, CSS, and JavaScript.

## Features

- Mobile hamburger menu
- Toggle dropdown using JavaScript
- Desktop horizontal layout
- Responsive design using media queries

## Technologies Used

- HTML
- CSS (Flexbox + Media Queries)
- JavaScript (DOM manipulation)

## How It Works

- Mobile: Links are hidden and toggled using a hamburger button.
- Desktop: Links are displayed horizontally and hamburger is hidden.
